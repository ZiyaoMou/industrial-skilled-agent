# Geely Quality Rootcause Copilot

Use this skill when the task is about defect triage, root cause discovery, and 8D/QRQC report drafting.

## Inputs
- Defect symptom and part name
- Production line, station, and shift information
- Defect counts by batch, supplier, and process parameters

## Workflow
1. Build a concise D2 problem statement.
2. Create an initial fishbone hypothesis map (man, machine, method, material).
3. Prioritize top probable causes by data confidence.
4. Propose containment and verification actions for D5 and D6.

## Output
- 8D markdown draft with owner and due dates
- Suggested immediate containment actions
- Verification checklist for next shift review
